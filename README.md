# Match

A simple no log resume-job description matching tool.

## Features

- Paste job description and resume text
- Upload `.txt` resume files (drag & drop supported)
- Shows matched skills, missing skills, and match score
- Provides improvement suggestions
- Dark mode toggle

## Files

- `index.html` - structure
- `style.css` - styles
- `script.js` - functionality

## Usage

1. Open `index.html` in your browser
2. Paste a job description
3. Upload or paste your resume
4. Click **Analyze**

## Tech

- Vanilla HTML, CSS, JavaScript
- No dependencies
- Works offline
